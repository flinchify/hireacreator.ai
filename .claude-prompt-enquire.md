Replace the `handleMessage` function and the "Message" button in `CreatorHeroActions` with an "Enquire" button that opens a modal/bottom sheet.

The Enquire modal should have:
- Creator name at the top ("Enquire about working with {name}")
- Your name (pre-filled from auth if logged in, otherwise required input)
- Your email (pre-filled from auth if logged in, otherwise required input)
- Budget range dropdown: "Select budget", "Under $500", "$500 - $1,000", "$1,000 - $5,000", "$5,000 - $10,000", "$10,000+"

- Send Enquiry button
- The form should work whether logged in or not (so businesses and agents can enquire without signing up)

### 2. Create API endpoint: POST /api/enquiry (app/api/enquiry/route.ts)

```ts

// Validates all required fields
```

Use Resend (already set up in the project). Send from noreply@send.hireacreator.ai (check existing Resend usage for the from address pattern).

Email to creator should look professional:
- Subject: "New enquiry from {name} on HireACreator"
- Body: formatted HTML with enquirer name, email, budget, project type, message
- Reply-to header set to the enquirer's email so creators can reply directly
