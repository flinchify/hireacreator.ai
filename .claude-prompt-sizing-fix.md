Fix link-in-bio template sizing and background issues. The templates look unprofessional on mobile. Buttons too small, background repeats, content doesn't fill screen.

## In components/link-in-bio-content.tsx:

### 1. Background image fix
Find the BgLayer component (renders creator's bg image). The background image should:
- Cover the full viewport: background-size: cover; background-position: center; background-repeat: no-repeat;
- Be FIXED so it doesn't scroll: background-attachment: fixed (or use position:fixed with inset-0)
- Show only ONCE, never repeat
- Have a strong dark overlay on top (bg-black/60 or bg-black/70) so text is readable

Search for BgLayer or bgType or bg_type and fix.

### 2. Link buttons sizing - match Linktree
Linktree buttons are full-width, tall (56-60px), with clear text. Our buttons are too small.

In the BioLinksSection component, update the link button styling:
- Full width: w-full
- Taller: py-4 min-h-[56px] (Linktree uses about 56px height)
- Larger text: text-base font-semibold (not text-sm)
- More padding: px-6
- Clear borders on dark backgrounds

For the standalone links (non-slider), change from:
```
px-5 py-4
```
to:
```
px-6 py-4 min-h-[56px] text-base
```

### 3. Card sizing for bento/grid templates
For templates that use grid cards (like bento), make cards bigger:
- Social icon cards: min-h-[80px] with larger icons (w-6 h-6 not w-4 h-4)
- Bio card: proper padding, text-base not text-sm
- All cards should fill available width

### 4. Remove excessive gaps
The huge gap between the last link and "View Full Profile" CTA. Reduce spacing:
- The CTA should follow naturally after the links
- Max gap between sections: space-y-4 (not space-y-6 or space-y-8)

### 5. Content should fill the phone screen
The content sits in the top 40% of the screen with empty space below. Fix:
- Remove max-w-[480px] or similar constraints on mobile — content should be full width
- The container should be at least min-h-screen with content spread to fill it
- Use flex flex-col justify-between or similar to push CTA toward bottom

### 6. Profile section improvements
- Avatar should be bigger on templates (at least w-20 h-20 on mobile)
- Name text larger: text-2xl font-bold
- Headline/category text: text-sm, not text-xs
- More contrast: if dark bg, text should be pure white, not grey
