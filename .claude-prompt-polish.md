- Delete the entire components/3d/ directory
- In components/homepage-content.tsx: remove the import of HeroFlow3D or HeroFlowDynamic and the section that renders it (the "How it works" / "From signup to delivery" 3D section)
- Remove @react-three/fiber, @react-three/drei, three, @types/three from package.json
- Run npm install after removing them
- This will fix any remaining Vercel peer dependency issues too
